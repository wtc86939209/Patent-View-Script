# Patent-View-Script 🚀
**Fusion 360 专利图自动化导出工具集**

本项目旨在帮助专利绘图师和工业设计师，通过 Python 脚本在 Autodesk Fusion 360 中快速生成符合各国外观专利审查标准的图片。

## 核心功能
* **多国标准覆盖**：目前已支持中国 6 视图导出（`CHINA 6视图.py`） 及欧盟视图导出（`欧盟图.py`）。
* **动态视角计算**：根据当前模型姿态，自动计算主视、俯视、侧视等坐标。
* **自定义参数**：支持自定义图片分辨率（如 1772x2598）、透视图缩放比例及渲染延迟，确保光影稳定。

## 如何使用
1.  **打开 Fusion 360**：进入“工具” -> “加载项” -> “脚本和加载项”。
2.  **创建新脚本**：新建一个 Python 脚本。
3.  **复制代码**：将本仓库中对应国家的 `.py` 代码内容复制到 Fusion 360 的脚本编辑器中。
4.  **配置路径**：在脚本代码中修改 `save_directory` 为你本地的保存路径。
5.  **运行**：点击运行，等待脚本自动完成截图。
